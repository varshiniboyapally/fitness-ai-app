# AI Fitness App (Built with AWS PartyRock)

## Overview
This is an AI-powered fitness application created using AWS PartyRock.  
The app generates personalized fitness suggestions, workout ideas, and wellness guidance using generative AI.


## Live Application 
(https://partyrock.aws/u/Indu18/iA4C3RvoW/Health-and-Wellness-Personal-Assessment-Guide)

## How to use
1. Click the “Live Application” link above  
2. The app will open in your browser (PartyRock)  
3. Enter your answer according to the options such as:
   - "Age, Weight, height, Gender, workout frequency, Food habits etc."
   then according to your answer the app will generate analysis like:
   - "Your Ideal Weight Range"
   - "Lifestyle Analysis - Pros and Cons"
   - "Health Risk Assessment"
   - "Personalized Health Action Plan"
     
4. The AI will generate personalized fitness guidance, diet, suggests preliminary health checkups instantly  



## Features
- AI-generated workout plans  
- Fitness and wellness recommendations  
- Interactive prompt-based AI responses  
- Easy-to-use web interface  


## Built With
- AWS PartyRock  
- Generative AI (Amazon Bedrock models)


## Purpose
This project was built as part of a learning exercise to explore how generative AI can be applied in health and fitness use cases.
